Heart Disease Prediction Overview

Our machine learning models use several health indicators to predict the likelihood of heart disease:

  • Age, Sex, Chest Pain Type
  • Resting Blood Pressure, Cholesterol Levels, Fasting Blood Sugar
  • Resting Electrocardiographic Results
  • Maximum Heart Rate Achieved
  • Exercise Induced Angina
  • ST Depression Induced by Exercise
  • Number of Major Vessels Colored by Fluoroscopy
  • Thalassemia

Machine Learning Algorithms Used

The following machine learning techniques are applied to improve the accuracy and performance of heart disease prediction:

  • Random Forest: A robust ensemble learning technique that combines multiple decision trees.
  • Genetic Algorithm: A technique used to optimize the parameters of our models, improving performance.
  • Bat Algorithm: A bio-inspired algorithm for optimizing solutions, useful in feature selection and hyperparameter tuning.
  • Bee Algorithm: Another nature-inspired algorithm that helps in optimizing the overall accuracy of the prediction model.

Prediction Accuracy and Insights

The heart disease prediction models achieve high accuracy, thanks to the use of advanced machine learning techniques:

  • Accurate predictions based on key health parameters
